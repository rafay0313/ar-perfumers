
  # Luxury E-commerce Website Design

  This is a code bundle for Luxury E-commerce Website Design. The original project is available at https://www.figma.com/design/1uYmmHKQ1ICmJ8fWKVLQ4n/Luxury-E-commerce-Website-Design.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
  Run `npm run server` to start the backend API on `http://localhost:4000/api`.

  Demo login credentials:
  - Admin: `admin@arluxe.com` / `admin123`
  - Customer: `user@arluxe.com` / `user123`
